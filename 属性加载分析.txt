1. Default 默认属性
   共享：添加到Manager的静态共享属性变量 TResInfo m_SharedResInfo
   私有：添加到Manager的成员私有属性变量 TResInfo m_ResInfo
   ****：TResInfo 添加默认属性的特点：重复添加时，后添加的属性取代先添加的属性
2. Include 引入的属性
   引入定义控件默认属性的文件，属性定义同 Default。
   跟 Default 相比，谁在前，先解析谁
3. Window 根节点下既有 Default ，又有 Include 时， Default 优先解析
   这种情况下，Default 属性的 shared 如果与 Include 文件中的一样，则会被覆盖。
4. 共享/私有 默认属性怎么添加到控件
   1. 添加默认属性:共享/私有 默认属性，优先选择私有，如果不存在则选择共享。即两者不能叠加。
   2. 添加具体属性，即与控件写在一起的属性
5. 默认属性使用总结
   1. 共享 默认属性，单独放到一个文件中，使用 Include 引入
   2. 私有 默认属性，放到窗体布局文件中，使用 Default 引入
   3. 如果一个窗体中，同一类控件，有不同的样式：则用 Default 引入使用较多的样式；其它样式直接写到具体用到控件的地方
6. 如何实现属性叠加，像下面说明的顺序添加属性
   1. 添加 共享 默认属性
   2. 添加 私有 默认属性，覆盖相同的属性
   3. 添加 具体 属性，覆盖相同的属性
